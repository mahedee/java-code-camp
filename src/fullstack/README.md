## Notes

- Pull mysql images : docker pull mysql
- Run mysql docker image: docker run --name mysqldb -e MYSQL_ROOT_PASSWORD=Password -e MYSQL_DATABASE=mydb -d -p 3306:3306 mysql:latest

- User TablePlus for database management
